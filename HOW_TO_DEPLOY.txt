📋 STEP-BY-STEP DEPLOYMENT (30 minutes)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: Install Dependencies (5 minutes)
────────────────────────────────────────

Open Terminal 1:
  cd client
  npm install

Open Terminal 2:
  cd server
  npm install

✅ Done when: No errors, "up to date" message appears

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 2: Start Application (2 minutes)
──────────────────────────────────────

In Terminal 1 (keep open):
  cd server
  npm start

In Terminal 2 (keep open):
  cd client
  npm run dev

✅ Done when: Both servers running, no errors

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 3: Test in Browser (10 minutes)
─────────────────────────────────────

1. Open: http://localhost:5173
2. Login with your credentials
3. Go to Reports section
4. Generate any report
5. Try exporting to:
   - CSV (should work)
   - PDF (should work, warning if >1000 rows)
   - Excel (should work with metadata)

✅ Done when: All exports work without errors

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 4: Run Automated Tests (5 minutes)
────────────────────────────────────────

In Terminal 3:
  ./test_critical_fixes.sh

✅ Done when: Tests show authentication required, SQL injection blocked

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
